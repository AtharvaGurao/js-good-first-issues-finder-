Good First Issues

This is a list of JavaScript repositories with good first issues for newcomers to open source. Contributions are welcome!

This list gets updated every day at midnight.

  • Make it easier to copy the version, revision, and OS when reporting an issue
  • Delete and search icons are overlapping in brave://history

cncf/landscape-graph

  • Create documentation site (GitHub Pages or Netlify) using Docusaurus

vercel/next.js

  • [Feature Request] View the app "On Your Network"
  • Parameter on AppType is used incorrectly
  • Warnings when debugging in VS Code
  • Add support for server-timing header
  • Docs: Caching in GitHub Actions with Bun

MetaMask/metamask-extension

  • Fix Button Alignment of Send Button in Nft-Details

Automattic/wp-calypso

  • Hever / Varia: Mobile Menu has a transparent background on a specific pages with a Contact Form
  • Theme Demo: Improve purchase bar
  • Login: cookie notice needs padding and eye icon isn't vertically centred

usebruno/bruno

  • Using dot (.) in variables names are recognized by the UI but not picked up when sending request

parcel-bundler/parcel

  • Social Icons block: Update the Medium logo
  • Scripts: Enhance the way entry points are detected in projects consisting of blocks and editor plugins
  • Block Registration: add SlotFill to extend a block's description
  • Scripts: Lazy loaded JavaScript files do not break browser cache upon changes
